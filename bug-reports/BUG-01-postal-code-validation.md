2. Добавить любой товар в корзину
3. Перейти в корзину и нажать "Checkout"

## Шаги воспроизведения

| Шаг | Действие |
|-----|----------|
| 1 | Перейти на страницу `/checkout-step-one.html` |

##  Ожидаемый результат

Появляется сообщение об ошибке: *"Postal Code must be 5-6 digits"* или форма не принимает значение и просит исправить.

---

##  Фактический результат

- Форма принимает 11 цифр без ошибки
- Переход на страницу `/checkout-step-two.html` (Overview) выполнен успешно

## 💡 Предполагаемая причина

На форме отсутствует клиентская и серверная валидация поля Postal Code:
- Нет проверки на длину
- Нет проверки на допустимые символы (только цифры)
