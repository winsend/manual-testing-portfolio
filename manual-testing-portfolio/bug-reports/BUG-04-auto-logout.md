## Шаги воспроизведения

| Шаг | Действие |
|-----|----------|
| 1 | Выполнить вход с валидными учетными данными: `standard_user` / `secret_sauce` |
| 2 | Добавить любой товар в корзину (например, Sauce Labs Backpack) |
| 3 | Оставить страницу открытой и **не совершать никаких действий** в течение 10 минут |
| 4 | По истечении 10 минут нажать на любую кнопку или обновить страницу |
| 5 | Проверить статус авторизации |

## Ожидаемый результат

| Проверка | Ожидание |

## Фактический результат

| Время | Событие |
|-------|---------|
| 0 мин | Вход выполнен успешно |
| 10 мин | Происходит автоматический выход из аккаунта |
| 10+ мин | Пользователь перенаправлен на страницу логина |
| 10+ мин | Корзина с добавленными товарами **сохранена** |
