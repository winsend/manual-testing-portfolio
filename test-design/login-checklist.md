## Предусловие для всех проверок

Открыта страница https://www.saucedemo.com/ в браузере (кэш и куки очищены)

---

## 1. Позитивные сценарии (Happy Path)

| № | Проверка | Ожидаемый результат | Статус |
|---|----------|---------------------|--------|
| 1.1 | Вход с `standard_user` / `secret_sauce` | Выполнен вход на главную страницу `/inventory.html` | + |
| 1.2 | Вход с `problem_user` / `secret_sauce` | Выполнен вход, видны баги (изображения, кнопки) | + |
| 1.3 | Вход с `performance_glitch_user` / `secret_sauce` | Выполнен вход (возможна задержка до 5 сек) | + |
| 1.4 | Вход с `error_user` / `secret_sauce` | Выполнен вход, видны ошибки отображения | + |
| 1.5 | Вход с `visual_user` / `secret_sauce` | Выполнен вход, видны визуальные искажения | + |
